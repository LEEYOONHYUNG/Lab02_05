2. Why do Github users need an SSH key pair? (Users can use either SSH key pair or github account info.) Please write down a brief explanation of SSH key.
SSH key는 서버 접속 시 비밀번호 대신 key를 제출하는 방식이다. 주로 비밀번호보다 높은 수준의 보안, 그리고 로그인 없는 서버 접속을 위하여 사용된다.
SSH key, 또는 SSH key pair은 공개키와 비공개키로 나뉘어져 구성된다. 
이때 공개키는 remote machine(SSH 서버)에 저장되고 비공개키는 local machine(SSH client)에 저장되며, 
SSH 접속 시도 시 SSH client가 로컬 머신의 비공개키와 원격 머신의 공개키를 비교해 일치하는지 보는 방식으로 접속 허가 여부를 판단한다.
Github 역시 remote machine에 유저가 각자의 기기에서 클라이언트로써 접속하는 방식이며, 
해당 체계를 채택하고 있기 때문에 Github 유저들은 SSH key를 생성하고 사용할 필요가 있는 것이다.
