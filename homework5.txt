2. Why do Github users need an SSH key pair? (Users can use either SSH key pair or github account info.) Please write down a brief explanation of SSH key.
In GitHub, SSH keys are used to allow users to connect to GitHub with SSH protocal. After generating SSH keys, users do not need to supply usernames and access token at every visit. (GitHub)
In general, an SSH key is an access credential for the SSH (secure shell protocol) network, which is used to provide secure connections over an unsecured network. A pair of SSH keys (pulic key & private key) is needed to initiate a secure handshake between remote computers.(Atlassian)
