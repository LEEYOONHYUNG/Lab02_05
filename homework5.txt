2. Why do Github users need an SSH key pair? (Users can use either SSH key pair or github account info.) Please write down a brief explanation of SSH key.
SSH key is an alternate way to identify Github user that doesn't require to enter github account info (username and password) every time. In other words, Github users need an SSH key pair in order to skip the cumbersome process of entering whole github account info each time.
