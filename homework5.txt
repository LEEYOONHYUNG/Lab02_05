2. Why do Github users need an SSH key pair? (Users can use either SSH key pair or github account info.) Please write down a brief explanation of SSH key.
SSH Key는 서버에 접속을 할 때 비밀번호 대용으로 사용되는 말 그대로 key이다. 보안 수준이 비밀번호보다 높고 로그인을 하지 않고 자동으로 서버에 접속할 수 있다는 장점이 있다. SSH key는 짝으로 작동하여 내 기기에 있는 key와 github에 있는 key가 맞으면 되는 것이다.
